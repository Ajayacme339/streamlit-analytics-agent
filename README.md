# Streamlit Analytics Agent

An **AI analytics agent** with a **Streamlit** front end — ask questions about your data in natural language and get analysis back.

## What's inside

```
Streamlit_app/
├── app.py             # the Streamlit application
└── requirements.txt   # dependencies
```

## Run locally

```bash
cd Streamlit_app
pip install -r requirements.txt
streamlit run app.py
```
